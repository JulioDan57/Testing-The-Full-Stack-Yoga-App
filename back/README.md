# Yoga App !


Pour exécuter les tests unitaires et produire automatiquement un rapport de couverture JaCoCo, utilisez la commande :
> mvn clean test

Pour exécuter l’ensemble des tests (unitaires + intégration) et générer les rapports de couverture JaCoCo, utilisez :
> mvn clean verify

Les rapports de couverture JaCoCo sont disponibles dans :
- Tests unitaires : back/target/site/jacoco/index.html
- Tests d’intégration : back/target/site/jacoco-it/index.html
